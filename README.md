# movie_playground
* github
```shell
https://github.com/thechanmoon/movie_playground.git
```


* Frontend Install
```shell
npm install
```

* Frontend Run
```shell
npm start
```

* Backend Install
```shell
bundle install
rails db:create db:migrate db:seed
```

* Backend Run
```shell
rails s
```
